# 使用方法
## 下载
1. 直接下载压缩包文件，然后解压到本地（推荐）.

2. 如果你安装了git，也可以使用git clone命令下载到本地.

```shell
git clone #
```
# 运行
通过npm安装本地服务第三方依赖模块(需要已安装Node.js)
```shell
npm install
```
启动MongoDB服务
```shell
mongod
```
启动node服务器
```shell
npm start
```
